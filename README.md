# Licence Plate Detection Service

This service implements a YOLOv8-based licence plate detection that takes one or more images as input and returns detected licence plates with bounding boxes, confidence scores, and cropped plate images.

_Check the [related documentation](https://docs.swiss-ai-center.ch/reference/services/licence-plate-detection/) for more information._

> [!NOTE]  
> The focus of this service is on integration with the Swiss AI Center service framework. Detection accuracy may vary depending on model quality and training data.
